[진행 상황]
팀 이름: 멀티캠퍼스유치원 문지기
팀 주제: smart doorman(외출 케어 서비스)
기능
 1. 얼굴 인식 판독과 문 개폐 기능
 - 문 밖 카메라 설치 후, 카메라를 통해 얼굴 인식과 판독
 - 예) 영상의 얼굴이 주인이라고 판독되면, 문 열어줌()
 2. 문 안쪽에 부착한 디스플레이 기능(태블릿)
 - 날씨 정보
 - 미세 먼지 정보
 - 택배 일정 정보와 알림 메시지 표시(송장번호를 통한?)
 - 전기? 보일러? 가스? 문 앞에서 on/off
 - 문 안쪽에 카메라 설치, 마스크 썼는지 판독후, 아니라면 알림 메시지 표시
 3. 샤오미 초인종 기능
 - 문 밖에서 인식되는 생물체가 x초이상 머무른다면 영상 자동 저장
 - 사용자(집 주인)에게 앱으로 알림 메시지 전송(영상 전송인지, 메시지 전송인지 구분 필요)
 4. 택배 보관함
 - 1) 택배 문자가 사용자에게 오면, 사용자는 앱을 통해 택배 보관함을 오픈 ->
   택배기사는 택배를 보관함에 넣음 -> 택배함에 부착된 무게 센서로 택배 감지
   -> 사용자에게 앱으로 알림 메시지 전송됨(택배 왔다~~)
 - 2) 택배 기사 얼굴 감지하면 택배 보관함 열어주기(택배기사정보db가 준비된다는 전제)
 3,4 는 문 밖 디스플레이(태블릿)에 두 가지 터치스크린(초인종/택배함)을 통해 입력함
 5. 충격 감지 기능
 - 충격 센서, 소리 센서로 큰 데시벨이나 큰 충격이 x초 반복되면 감지하여 앱으로 알림 메시지 전송
시나리오
1. 안면인식 개폐
2. 문 안쪽 디스플레이
- 날씨, 미세먼지, 택배알림, 마스크, 집안 전기?... on/off
3. 샤오미 초인종 -> 어플로 알림 메시지 전송
4. 택배 보관함
- 1 사용자가 어플로 열어준다
- 2 안면인식으로 열린다
5. 충격, 진동 감지 -> 어플로 알림 메시지 전송
